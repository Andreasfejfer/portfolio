# Portfolio

## Local Setup (Node.js)

This project uses JavaScript modules and should be worked on with Node.js installed.

Recommended:
- Node.js `20 LTS`
- `nvm` for version management

Install with `nvm`:

```bash
# install nvm (if missing)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# restart shell, then install/use Node 20
nvm install 20
nvm use 20
nvm alias default 20

# verify
node -v
npm -v
```
